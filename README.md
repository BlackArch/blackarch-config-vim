## Description

BlackArch Linux configuration files for vim editor.

## Installation

`pacman -S blackarch-config-vim`

## Configuration

Copy the following files:

1. `cp -a /usr/share/blackarch/config/vim/vim ~/.vim`
2. `cp -a /usr/share/blackarch/config/vim/vimrc ~/.vimrc`
